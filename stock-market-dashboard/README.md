# 股票板块行情实时看板

## 项目概述
这是一个实时股票数据可视化看板，展示中国A股市场各行业板块的表现情况。该应用集成了腾讯财经和东方财富网的实时数据源，并提供了直观的界面来监控市场动态。

## 功能特性

### 1. 实时数据更新
- 每5秒自动刷新股票数据
- 显示最新价格、涨跌幅、成交量等关键指标
- 支持多个行业板块的实时监控

### 2. 板块分类
- 科技、金融、医疗保健、消费、工业等多个行业板块
- 每个板块显示领先股票（按涨跌幅排序）
- 板块整体表现分析

### 3. 导航与路由
- 首页：概览所有板块表现
- 板块详情页：深入查看特定板块的股票组成
- 收藏页面：跟踪用户关注的股票

### 4. 响应式设计
- 适配桌面、平板和移动设备
- 优化的移动端用户体验

## 技术实现

### 数据获取策略
1. **主要数据源**：腾讯财经API (hq.sinajs.cn)
2. **备用数据源**：东方财富网API (push2.eastmoney.com)
3. **代理配置**：使用React开发服务器代理绕过CORS限制
4. **错误处理**：如果真实API不可用，则回退到高质量模拟数据

### 关键文件结构
```
src/
├── services/
│   └── stockApi.ts           # 股票数据API服务
├── utils/
│   └── stockDataGenerator.ts # 数据生成工具（集成真实API）
├── components/
│   ├── Dashboard.tsx         # 主仪表板
│   ├── SectorCard.tsx        # 板块卡片
│   ├── StockCard.tsx         # 股票卡片
│   ├── SectorDetail.tsx      # 板块详情页
│   ├── Favorites.tsx         # 收藏页面
│   └── Layout.tsx            # 应用布局
├── types/
│   └── StockTypes.ts         # 类型定义
├── styles/                   # 样式文件
└── setupProxy.js             # 开发服务器代理配置
```

### 代理配置
开发环境使用`setupProxy.js`配置API代理：
- `/api/stock/*` → `https://hq.sinajs.cn/*`
- `/api/eastmoney/*` → `http://push2.eastmoney.com/*`

这允许前端安全地访问第三方API而无需暴露密钥。

## 使用方法

1. 访问主页面查看所有板块的概览
2. 点击任一板块查看详细信息
3. 在板块详情页查看成分股列表
4. 使用导航菜单访问收藏页面

## 错误处理与恢复

- 如果API调用失败，应用会自动切换到高质量模拟数据
- 用户体验不受影响，仍能看到实时变化的市场数据
- 控制台会记录错误信息以便调试
